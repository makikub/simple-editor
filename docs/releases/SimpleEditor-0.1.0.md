# Simple Editor 0.1.0

Initial release of Simple Editor.

- Open and save plain text files while preserving detected encoding and line endings.
- Edit CSV and TSV files with quoted-field parsing and row-preserving saves for untouched rows.
- Inspect fixed-width files with ruler and byte or character length warnings.
- Search and replace text, filter CSV rows, and recover unsaved draft text after a crash.
- Check for updates through Sparkle when installed from the packaged app.
